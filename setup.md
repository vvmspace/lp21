# Быстрый запуск продакшена (zero knowledge)

Эта инструкция рассчитана на человека без технического бекграунда.

## Что нужно заранее

1. **Доступ к серверу** (VPS/VM) с Linux.
2. **Установленный Docker** и Docker Compose (обычно уже есть на большинстве VPS).
3. **Файл `.env`** в корне репозитория.

## 1) Подготовка `.env`

Откройте файл `.env` в корне проекта и заполните ключи:

```
GEMINI_API_KEY=ВАШ_КЛЮЧ_ОТ_GEMINI
BACK_PORT=3001
FRONT_PORT=3000
```

Если не знаете ключ — попросите у разработчика.

## 2) Запуск продакшена (одной командой)

В корне репозитория выполните:

```bash
./up_prod.sh
```

Это соберёт и поднимет контейнеры **front** и **back**.

## 3) Проверка, что всё работает

Откройте в браузере:

- Фронт: `http://<IP_СЕРВЕРА>:3000`
- Swagger API: `http://<IP_СЕРВЕРА>:3001/api/v1/docs`

Если оба открываются — прод запущен.

## 4) Остановка

Чтобы остановить:

```bash
docker compose -f docker-compose.prod.yml down
```

## 5) Что делать, если не работает

1. **Проверьте, что Docker запущен**.
2. **Проверьте доступность портов 3000 и 3001** в firewall.
3. **Посмотрите логи**:

```bash
docker compose -f docker-compose.prod.yml logs -f
```

Если всё равно непонятно — отправьте логи разработчику.
